# Support

## Comment obtenir de l'aide

Les repos Curious Factory sont **privés** et destinés à un usage interne. Pas de support public.

### Pour l'équipe interne (Emmanuel, Thibaut, Nolly)

| Type de question | Où aller |
|---|---|
| Bug rencontré dans un produit | Notion → 🍹 VIP Arena → 📋 Tickets → bouton **🐛 Reporter un bug** |
| Idée de feature | Notion → 🍹 VIP Arena → 📋 Tickets → bouton **💡 Proposer une idée** |
| Question process / setup / archi | Slack interne ou direct à Emmanuel |
| Audit / qualité code | Notion → 🍹 VIP Arena → 🛠️ Dev & Infra → audit pipeline |

### Pour les externes

Les produits Curious Factory (Quiz Boxing, VIP Arena) sont commercialisés en B2B. Contacter **contact@curiousfactory.fr** pour toute demande commerciale ou support client.
